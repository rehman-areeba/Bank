Your password MUST contain:

===============================================
TROUBLESHOOTING
===============================================

❌ "Registration failed" error:
   → Check browser DevTools Console for details
   → Check Network tab for API response
   → Ensure backend is running

❌ Password validation errors:
   → Make sure password meets ALL requirements
   → Check the hint below password field
   → Try: Password123!

❌ "Email already registered":
   → Use a different email
   → Or delete user from database:
     DELETE FROM Users WHERE Email = 'john@example.com';
     DELETE FROM Accounts WHERE UserId IN (SELECT Id FROM Users WHERE Email = 'john@example.com');

❌ Backend error 500:
   → Check backend console for error details
   → Ensure database is running
   → Check connection string in appsettings.json
