USAGE INSTRUCTIONS
===============================================

1. START APPLICATION:
   - Double-click START.bat
   OR
   - Terminal 1: cd BankingApi && dotnet run
   - Terminal 2: cd banking-ui && npm run dev

2. REGISTER USER:
   - Go to http://localhost:5173
   - Click "Register here"
   - Fill form with Password123! (or similar)
   - Submit

3. VIEW DASHBOARD:
   - See your default Savings account
   - View recent transactions (empty initially)

4. CREATE NEW ACCOUNT:
   - Click "New Account" button
   - Select account type
   - Submit

5. ADD BALANCE (for testing):
   - Use SQL script: ADD_BALANCE.sql
   - Or use Admin panel (if Admin role)

6. MAKE TRANSFER:
   - Click "Transfer Money"
   - Select source account
   - Enter recipient account number
   - Enter amount
   - Confirm

7. VIEW TRANSACTIONS:
   - Click "View Transactions"
   - Apply filters if needed
   - Load more for pagination

8. ADMIN PANEL (Admin only):
   - Click "Admin Panel" button
   - View all accounts
   - Freeze/unfreeze accounts
   - View audit logs
   - Monitor security
